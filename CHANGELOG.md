# Changelog

## 1.0.0

- Initial release of Mogu — a native macOS GUI for Mole.
- Status: live CPU, memory, disk, network, battery, top process monitoring, and total space-freed summary.
- Clean: preview and execute deep cleanup of caches, logs, and browser leftovers.
- Uninstall: multi-select app removal with leftover file preview, preview-before-delete enforcement, protected app admin flow, and Homebrew cask safeguards.
- Installer: read-only scan for leftover `.dmg`, `.pkg`, `.zip`, and related installer files.
- History: timeline of cleanup, purge, optimize, and uninstall sessions from Mole history.
- Optimize: one-click system optimization with admin fallback for protected operations.
- Analyze: visual disk usage breakdown with large-file detection and custom scan paths.
- Purge: project build artifact cleanup for `node_modules`, `target`, `.build`, and similar directories.
- Permissions: minimal permission model explainer for Administrator and optional Full Disk Access.
- Menu bar: lightweight system health widget with quick actions.
